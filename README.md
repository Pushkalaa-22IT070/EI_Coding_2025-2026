   **Factory Pattern** - Car Factory
   
   **Singleton Pattern** - Printer Spooler
3. Two use cases to demonstrate two structural design pattern.
   
   **Adapter Pattern** - Media Player
   
   **Bridge Pattern** - Remote Control
